How to Use This Dashboard

Dashboard Control

Use the toggle switch in the header to enable/disable the entire dashboard. When disabled, data updates will pause.

Real-Time Monitoring

The dashboard updates every 2 seconds with new sensor readings. Charts display the last 20 data points for trend analysis.

Alerts System

Automatic alerts are triggered when sensor values exceed thresholds. Critical alerts appear as toast notifications.

Export Data

Click the "Export to CSV" button in the data table section to download sensor readings for offline analysis.
